# User login API Endpoint

## Specs

**Path:** `/login`

**Method:** `POST`

**Request:**

| field  | type         | status   | description              |
| ------ | ------------ | -------- | ------------------------ |
| mobile | Phone Number | required | The user's mobile number |

**Response:**

<!--
| field | type | status | description |
| ----- | ---- | ------ | ----------- |
-->

**Possible Errors:**

| code | Condition     |
| ---- | ------------- |
| 400  | Invalid login |
